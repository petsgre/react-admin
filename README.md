# React-admin

基于 React 的后台管理系统模板，帮助企业快速生成后台管理系统

## 基本功能

- 登录(普通用户,密码可见设置) /
- 个人中心,个人设置 /
- 角色（管理员，普通），权限（模块权限）/
- 仪表盘(d3) /
- 全屏 F11 /
- 国际化（Ch,Eh）/
- 错误页,抛错(全局监听) /
- 示例 icon /
- 多页 tab 切换 /
- 面包屑导航 /
- 表格(多种) /
- extend 功能组件(逐步扩充) etc.
